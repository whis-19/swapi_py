TASK 1: Docker Containerization
Containerize the application using Docker.
Requirements:
•

Create a Dockerfile for the application.

•

Build a Docker image.

•

Run the container locally.

•

Verify that the application runs correctly inside the container.

TASK 2: Multi-Container Setup using Docker Compose
Create a Docker Compose configuration to run multiple services.
The compose configuration should include:
•

Frontend container

•

Backend container

•

Database container

Requirements:
•

Define multiple services

•

Configure container networking

•

Use environment variables where necessary

•

Map container ports to host ports

Deployment

•
